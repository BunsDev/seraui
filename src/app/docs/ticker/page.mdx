# Number Ticker
A smooth animated number counter component that animates from 0 to any target value with customizable formatting and styling.

## Usage

```tsx
import NumberTicker from "./ticker"

export default function Example() {
  return (
    <NumberTicker
      value={100}
      duration={2500}
      className="text-4xl font-bold"
      prefix="$"
      decimalPlaces={2}
    />
  )
}
```

## Examples

### Basic Counter
```tsx
<NumberTicker value={42} />
```

### Currency Display
```tsx
<NumberTicker 
  value={1234.56} 
  prefix="$" 
  decimalPlaces={2}
  className="text-2xl font-bold text-green-600"
/>
```

### Percentage with Callback
```tsx
<NumberTicker 
  value={95.5} 
  suffix="%" 
  decimalPlaces={1}
  duration={3000}
  onComplete={() => console.log('Animation complete!')}
/>
```
