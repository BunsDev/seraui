### Table Usage Guide

The Table component supports multiple variants for different use cases:

- **Default**: Clean, professional look with subtle shadows
- **Striped**: Alternating row colors with gradient header
- **Bordered**: Clear borders for data-heavy tables
- **Minimal**: Clean, borderless design for simple data

```jsx
// Basic usage
<Table
  headers={["Name", "Role", "Status"]}
  rows={[
    ["John Doe", "Developer", "Active"],
    ["Jane Smith", "Designer", "Active"]
  ]}
/>

// With variant
<Table
  variant="striped"
  headers={["API", "Status", "Response"]}
  rows={[
    ["/users", "✅ Active", "45ms"],
    ["/posts", "⚠️ Slow", "890ms"]
  ]}
/>
```
